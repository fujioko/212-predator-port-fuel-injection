This is the final version of the code as of the end of Robot Cantina, Season 6 Episode 12. It is intended to run on an Arduino UNO.
This project and its code are experimental. I take no responsibility for any issues. Testing was performed on a 212cc Predator engine, both unloaded and under a heavy load (RPM range 1,400–3,600). The load device applied substantial resistance but did not absorb the engine’s full 6.5 HP. The code is provided for viewing only; I will not maintain it or provide technical support. Use at your own risk.
